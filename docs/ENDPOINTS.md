## Bíblia (schema relacional — Sprint 4.1)

| Método | Rota | Descrição |
|--------|------|-----------|
| GET | `/api/bible/books` | Lista livros (`?testament=AT\|NT`) |
| GET | `/api/bible/{slug}/{chapter}` | Capítulo completo com navegação prev/next |
| GET | `/api/bible/{slug}/{chapter}/{verse}` | Versículo individual |
| GET | `/api/bible/search?q=...` | Busca unificada (referência + FTS) |

**Exemplo:** `GET /api/bible/genesis/1`

```json
{
  "book": { "slug": "genesis", "name": "Gênesis", "abbreviation": "Gn" },
  "chapter": { "number": 1, "verses_count": 31 },
  "verses": [
    { "number": 1, "text": "No princípio, Deus criou o céu e a terra." }
  ],
  "navigation": {
    "prev": null,
    "next": { "book_slug": "genesis", "chapter": 2 }
  }
}
```

**Busca:** `GET /api/bible/search?q=Jo%C3%A3o+3%2C16`

```json
{
  "type": "reference",
  "slug": "joao",
  "chapter": 3,
  "verse_start": 16,
  "results": [
    { "verse": 16, "text": "Porque Deus amou o mundo...", "book": { "slug": "joao", "name": "João", "abbrev": "Jo" } }
  ],
  "total": 1
}
```

## Schema SQL

### daily_gospel (expandido Sprint 3)

```sql
create table public.daily_gospel (
  id bigint generated by default as identity primary key,
  date date not null unique,
  referencia text not null,
  texto text not null,
  liturgical_season varchar(20),
  sunday_cycle char(1),
  ferial_cycle char(2),
  week_number smallint,
  pericope varchar(40),
  book_abbrev varchar(10),
  liturgical_key varchar(30),
  created_at timestamp with time zone default now(),
  updated_at timestamp with time zone default now()
);
```

### Schema relacional da Bíblia (Sprint 4.1)

```sql
-- Versões/traduções
create table public.bible_versions (
  id bigint generated by default as identity primary key,
  slug varchar(20) not null unique,
  name varchar(100) not null,
  language varchar(10) not null default 'pt',
  is_default boolean not null default false
);

-- Livros (por versão)
create table public.bible_books (
  id bigint generated by default as identity primary key,
  version_id bigint not null references public.bible_versions(id),
  testament char(2) not null check (testament in ('AT', 'NT')),
  abbreviation varchar(10) not null,
  slug varchar(60) not null,
  name varchar(100) not null,
  position smallint not null,
  chapters_count smallint not null default 0,
  unique(version_id, abbreviation),
  unique(version_id, slug)
);

-- Capítulos
create table public.bible_chapters (
  id bigint generated by default as identity primary key,
  book_id bigint not null references public.bible_books(id),
  number smallint not null,
  verses_count smallint not null default 0,
  unique(book_id, number)
);

-- Versículos com Full Text Search
create table public.bible_verses (
  id bigint generated by default as identity primary key,
  book_id bigint not null references public.bible_books(id),
  chapter_id bigint not null references public.bible_chapters(id),
  number smallint not null,
  text text not null,
  search_vector tsvector generated always as (to_tsvector('portuguese', text)) stored,
  unique(chapter_id, number)
);

create index idx_bible_verses_fts on public.bible_verses using gin (search_vector);
```

## Regras litúrgicas migradas do Laravel

### Ciclos dominicais
- Ano A: Evangelho de Mateus (2023, 2026...)
- Ano B: Evangelho de Marcos (2024, 2027...)
- Ano C: Evangelho de Lucas (2025, 2028...)
- Fórmula: `ano_dominical = (ano - 2) % 3` → 0=A, 1=B, 2=C

### Ciclo ferial (ano I/II)
- Ano ímpar → Ano I
- Ano par → Ano II

### Cores litúrgicas
- **Verde**: Tempo Comum
- **Roxo**: Advento, Quaresma
- **Branco**: Natal, Tempo Pascal, festas do Senhor/Maria/Santos não mártires
- **Vermelho**: Pentecostes, mártires, Sexta-Feira Santa

### Cálculo da Páscoa (algoritmo de Gauss)
Implementado fielmente em `liturgical_lib.py` conforme original `LiturgicalCalendar.php`.

### Santos (90+ entradas)
Arquivo `saints_calendar.py` com data, nome e descrição curta, conforme `SaintsCalendar.php`.

### Lecionário Dominical (~170 entradas)
Arquivo `lectionary_data.py` mapeando chave litúrgica → (perícope, abreviação do livro), conforme `lectionary.php` + `daily_gospels` migration.
